# KO Localization Checklist

## 범위

- 설정창 라벨/설명
- Notice/에러 문구
- README_KO 및 운영 문서

## 스모크 체크

1. 설정창 탭 라벨이 정상 표시된다.
2. `Login (OAuth)` 탭에서 필수 입력 안내가 한국어로 자연스럽다.
3. OAuth 실패 문구가 원인별로 구분되어 보인다.
4. `Parser` 탭의 HWP/PDF 안내 문구가 깨지지 않는다.
5. 채팅 최소 UI 안내 문구가 한국어로 자연스럽다.

## 회귀 체크

- 문자열 깨짐(모지바케) 없음
- 한/영 라벨 대응 누락 없음
- 동일 기능 라벨의 번역 불일치 없음
- 오류 메시지에 민감정보(토큰/키) 노출 없음

## 승인 기준

- 한국어 사용자 기준으로 설정 흐름을 문서 없이도 따라갈 수 있음
- 실패 시 다음 액션이 1~2문장 내에 제시됨
- README_KO와 실제 UI 동작이 일치함
